    MutableArithmetics.mutable_buffered_operate! — Method.
    mutable_buffered_operate!(buffer, op::Function, args...)

    Modify the value of args[1] to be equal to the value of op(args...), possibly modifying buffer. Can only be called if mutability(args[1], op, args...) returns true.

    source
    MutableArithmetics.mutable_buffered_operate_to! — Function.
    mutable_buffered_operate_to!(buffer, output, op::Function, args...)

    Modify the value of output to be equal to the value of op(args...), possibly modifying buffer. Can only be called if mutability(output, op, args...) returns true.
