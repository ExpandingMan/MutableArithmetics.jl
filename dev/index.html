    MutableArithmetics.mutability — Method.
    mutability(T::Type, ::typeof(op), args::Type...)::MutableTrait

    Return IsMutable to indicate that op(a::T, ::args[1], ...) returns a. That is, the result of the operation is stored in a and then a is returned. Equivalently, returns whether op_impl is supported.
